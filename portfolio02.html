Skills: TypeScript, Kaboom.js, Tiled

Utilizing TypeScript, Kaboom.js, and Tiled to Build a Metroidvania Style Game

Building a 2D platformer game with Kaboom.js, inspired by Metroidvania-style games, adds a rich layer of complexity to the gaming experience. This project focuses on creating a dynamic and interactive world where players navigate through interconnected rooms in an attempt to escape a factory. With Kaboom.js as the core game engine, the project incorporates room transitions, player controls, and visual cues to guide players through the game. The game uses TypeScript to manage state and define the interactions between the player and various game elements, including enemies and room exits.

This project shares several Metroidvania characteristics, such as map exploration, challenging platform mechanics, and player progression. Through dynamic scene management and custom-built notification systems, the game keeps players engaged. The development process covers core game mechanics like enemy AI, room transitions, and boss battles, making it an exciting and rewarding game development experience. This portfolio project showcases expertise in game design and front-end JavaScript development.
